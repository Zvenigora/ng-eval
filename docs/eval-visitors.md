# Eval Classes
Ng-eval provides a set of visitors for evaluating expressions. The visitors for following Acorn nodes are available:

- `BinaryExpression`,
- `Identifier`,
- `Literal`,
- `CallExpression`,
- `AwaitExpression`,
- `ConditionalExpression`,
- `MemberExpression`,
- `ArrayExpression`,
- `UnaryExpression`,
- `LogicalExpression`,
- `ThisExpression`,
- `NewExpression`,
- `TemplateLiteral`,
- `TaggedTemplateExpression`,
- `ObjectExpression`,
- `AssignmentExpression`,
- `UpdateExpression`,
- `ArrowFunctionExpression`.
